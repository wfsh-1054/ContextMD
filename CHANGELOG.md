# Changelog

All notable changes to the **ContextMD** project will be documented in this file.

## [1.1.1] - 2024-05-21

### Added
- **Auto Language Detection**: The app now automatically detects the user's browser language (English, Traditional Chinese, Simplified Chinese, Japanese, Korean) on the first visit.

### Fixed
- **Light Mode Contrast**: Fixed an issue where code block text was difficult to read in Light Mode due to low contrast. Code blocks now have distinct styling for both light and dark themes.

## [1.1.0] - 2024-05-20

### Changed
- **Rebranding**: Renamed application from "Markdown Live" to **ContextMD**.
- **UI Update**: Updated header styling and application title.
- **Documentation**: Updated `README.md` to reflect the new name and focus on LLM Context Engineering.

### Added
- **Internationalization (i18n)**: Added full translation support for UI elements.
- **Theme System**: Enhanced dark/light mode toggle with system preference sync.

## [1.0.0] - Initial Release

- Basic Markdown editing and preview.
- Split-pane view.
- GitHub Flavored Markdown support.
- JSON String Export tool for AI prompting.
